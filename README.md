## Descripción

Aplicación web desarrollada en React para el Segundo Parcial de Laboratorio II.
El proyecto implementa navegación entre páginas, gestión de publicaciones mediante formularios, uso de hooks de React y una arquitectura organizada por componentes, páginas, hooks y servicios.


## Hooks implementados

* useState
* useContext
* useRef

## Funcionalidades implementadas

* Navegación entre páginas mediante React Router DOM.
* Creación de publicaciones.
* Edición de publicaciones.
* Visualización de detalles de una publicación.
* Gestión de estado mediante Hooks de React.
* Compartición de estado utilizando Context API.
* Enfoque automático de elementos mediante useRef.
* Arquitectura modular organizada en componentes, páginas, hooks y servicios.


## Instalación


1. Clonar el repositorio:
git clone https://github.com/FedeAMasin/ParcialReact.git

2. Ingresar a la carpeta del proyecto:
cd ParcialReact

3. Instalar las dependencias necesarias:
npm install
Este comando descargará automáticamente todas las librerías requeridas para ejecutar la aplicación.

## Ejecución

Para iniciar el proyecto en modo desarrollo, ejecutar:
npm run dev

Una vez iniciado,Abrir en el navegador para acceder a::
http://localhost:5173
