Preparacion de ambiente para Poder visualizar el entorno.

1- ubicamos el comprimido con el nombre ParcialReact y lo descomprimimos en una ubicacion de nuestra preferencia
2- abrimos una consola de comandos y tipeamos la palabra cd
3- volvemos al explorador de archivos y copiamos la ruta a la carpeta que descomprimimos ( debemos fijarnos que la ruta apunte adentro de la carpeta que descomprimimos, no afuera de ella)
4-apretamos enter para posicionarnos sobre esa ubicacion.
5- escribimos el comando npm install ( en caso que usemos el paquete instalador de dependencias de node, sino adaptarlo al gestor de paquetes que usemos)
6- tipeamos el comando "npm run dev" y damos enter.
7- en la consola deberia mostrarlos una direccion "localhost"podemos clickear en ella para poder verificar el proyecto corriendo en local.
